# phishing-analysis
Structured analysis reports of real-world phishing samples

Static analysis reports of real-world phishing samples: header forensics and authentication results (SPF/DKIM/DMARC), URL and sending-infrastructure analysis, and attachment examination with oletools — no detonation, all analysis performed in an isolated VM. Each report ends with a verdict, defanged IOC table, and detection opportunities a mail gateway or SIEM could act on.

Status: In progress — first samples being worked.
